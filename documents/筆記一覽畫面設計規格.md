# Kotobase — 筆記一覽畫面設計規格

> 本文件提供給設計師，用於重新設計筆記一覽畫面。
> 說明各元素功能、互動行為，以及手機版的配置建議。

---

## 設計師注意事項

- 此畫面元素較多，手機版請特別注意**篩選功能的收納方式**，避免畫面擁擠。以下為建議，設計師可依整體設計調整：
  - 搜尋欄位 filter、分類複選篩選、行事曆篩選等，建議收在圖標按鈕後面，點擊後以 Bottom Sheet 或展開區塊呈現。
  - 分類快速按鈕建議使用**水平捲動**排列，不換行。
  - 排序建議使用 **Segmented Control**（iOS 分段控制器）。
- 筆記卡片的內容預覽行數、卡片高度等，請依設計稿自行調整。
- 所有視覺樣式請依 Figma 設計稿重新定義，本文件僅描述元素與功能。

---

## 畫面概述

筆記一覽是使用者瀏覽、搜尋、篩選所有已儲存筆記的主要畫面。提供多種篩選條件（關鍵字、分類、日期）幫助使用者快速找到目標筆記。

---

## 元素清單

### 1. 搜尋欄

- **類型：** Text Input（搜尋輸入框）
- **說明：**
  - 輸入關鍵字即時篩選筆記列表。
  - 搜尋範圍可透過「搜尋欄位 filter」（元素 3）設定要比對哪些欄位。
- **行動版：** 同網頁版。

---

### 2. 排序選擇器（最新/最舊）

- **類型：** Segmented Control / Select
- **說明：**
  - 切換筆記排列順序：最新到最舊 / 最舊到最新。
  - 依據筆記的建立時間排序。
- **行動版建議：** 使用 iOS Segmented Control，只有兩個選項，簡潔不佔空間。

---

### 3. 搜尋欄位 filter

- **類型：** Checkbox 勾選清單
- **說明：**
  - 讓使用者選擇搜尋時要比對的欄位，包含：標題、分類、子分類、日文內容、備註。
  - 提供「全選」與「清除」快速操作。
  - 預設全部欄位勾選。
- **行動版建議：** 收在搜尋欄旁的篩選圖標按鈕後面，點擊後以 Bottom Sheet 展開勾選。

---

### 4. 分類篩選器（分類 + 子分類複選）

- **類型：** 多選篩選清單
- **說明：**
  - 可複選多個主分類及子分類來篩選筆記。
  - 選取「全部」時顯示所有筆記；選取特定分類時只顯示該分類下的筆記。
  - 子分類篩選為進一步細化，可在已選主分類範圍內再篩選特定子分類。
  - 已啟用篩選時，按鈕上顯示徽章（已選數量）提示使用者目前有篩選條件。
- **行動版建議：** 以 Bottom Sheet 呈現，主分類與子分類分組顯示，子分類縮排於所屬主分類下方。

---

### 5. 分類快速按鈕

- **類型：** Pill Buttons（藥丸形按鈕列）
- **說明：**
  - 水平排列所有主分類按鈕，另有一個「全部」按鈕。
  - 點擊可快速切換顯示該分類的筆記，支援多選。
  - 選中狀態有明顯的視覺區別（顏色、發光等）。
- **行動版建議：** 使用水平捲動排列，不換行，手指左右滑動瀏覽。

---

### 6. 行事曆篩選器

- **類型：** Mini Calendar（迷你月曆）
- **說明：**
  - 以月曆形式顯示，可切換上一月 / 下一月。
  - 有筆記的日期會標示小圓點提示。
  - 點擊日期可篩選該日建立的筆記，支援多選日期。
  - 提供清除按鈕，顯示目前已選日期數量。
- **行動版建議：** 預設隱藏，放在日曆圖標按鈕後面，點擊後展開（Bottom Sheet 或 inline 展開皆可）。

---

### 7. 筆記一覽（列表區域）

- **類型：** Card List / TableView
- **說明：**
  - 顯示篩選後的所有筆記，以卡片形式排列。
  - 點擊卡片進入筆記詳細頁面。
  - 無符合結果時顯示空狀態提示文字。
  - 網頁版為多欄網格（依螢幕寬度 1～4 欄），手機版由設計師決定排版方式。

---

### 8. 筆記卡片 UI

- **類型：** Card（卡片元件）
- **說明：** 每張卡片顯示以下資訊：

| 欄位 | 說明 |
|------|------|
| 分類色條 | 卡片頂部的色條，顏色對應該筆記的主分類顏色 |
| 主分類標籤 | 顯示主分類名稱，背景色同分類顏色 |
| 子分類標籤 | 若有子分類則顯示，可能有多個 |
| 建立時間 | 筆記的建立日期時間 |
| 標題 | 筆記標題，超出範圍截斷 |
| 內容預覽 | 日文內容的前幾行預覽，行數與截斷方式請設計師調整 |

---

## 網頁版 vs 行動版差異

| 項目 | 網頁版 | iOS 行動版 |
|------|--------|------------|
| 篩選功能展開方式 | 直接顯示在頁面上，手機尺寸可收合 | 收在圖標按鈕後，以 Bottom Sheet 等方式展開 |
| 排序 | 下拉選單 | Segmented Control |
| 分類快速按鈕 | 自動換行排列 | 水平捲動不換行 |
| 行事曆 | 直接顯示 | 預設隱藏，點擊圖標展開 |
| 筆記卡片排列 | 多欄網格（1～4 欄） | 由設計師決定 |
| 功能範圍 | 全部保留 | 全部保留 |
